HTML 섹셔닝 요소(sectiong element)

ARIA 랜드마크를 기본적으로 정의하는 많은 HTML 섹셔닝 요소를 (예를 들어, main, nav, aside ...) 이해하는 것이 중요합니다. HTML 섹셔닝 요소가 연관된 랜드마크 구조에 대한 이해 없이 사용된다면, 보고 기술 사용자들은 콘텐트에 접근하고 웹 페이지와 상호작용 하는데 유효하지 못하고 혼동 될 소지가 가장 높습니다.

HTML 요소 기본 랜드마크 역할(role)
aside complementary
footer body 엘리먼트의 컨텍스트에 있는 경우 contentinfo. footer 엘리먼트는 다음 HTML 섹셔닝 요소(sectioning element)의 후손인 경우, contentinfo 랜드마크가 아닙니다:
  • article
  • aside
  • main
  • nav
  • section
form aria-labelledby, aria-label나 title 어트리뷰트를 사용하여 접근가능한 이름을 가지는 경우 form
header body 엘리먼트의 컨텍스트에 있는 경우 banner . header 요소는 다음 HTML 섹셔닝 요소(sectioning element)의 후손인 경우, banner 랜드마크가 아닙니다:
  • article
  • aside
  • main
  • nav
  • section
main main
nav navigation
section aria-labelledby, aria-label 나 title 어트리뷰트를 사용하여 접근가능한 이름을 가지는 경우 region
